X GEO FILTER GUARD - END-USER QUICK START

Goal:
Filter feed noise on X by country/region, with optional auto-mute or auto-block.

Available modes:
- Hide/Dim posts only (no account action)
- Auto mute matching accounts
- Auto block matching accounts

==================================================
1) DOWNLOAD
==================================================
Open:
https://github.com/kappa9999/x-geo-filter-extension/releases/latest

Download the ZIP release and unzip it.

==================================================
2) INSTALL IN CHROME
==================================================
1. Open Chrome
2. Go to: chrome://extensions
3. Enable Developer mode (top-right)
4. Click Load unpacked
5. Select folder: x-geo-filter-extension

Visual references:
- Popup: docs/screenshots/popup.png
- Settings: docs/screenshots/options-page.png

==================================================
3) CONFIGURE
==================================================
1. Open extension popup
2. Click Open settings
3. Add include rule(s), for example:
   - india

Choose a mode:

MODE A - Hide/Dim only
- Post behavior = Hide (or Dim)
- Auto action = None

MODE B - Auto mute
- Auto action = Mute
- Dry run = On first

MODE C - Auto block
- Auto action = Block
- Dry run = On first
- Max actions/hour = 1 or 2
- Delay = 3000ms
- Sightings before action = 2

4. Click Save settings
5. Refresh x.com and scroll your feed

==================================================
4) EMERGENCY STOP
==================================================
From popup:
- Disable extension
- Set Auto action to None
- Pause 30 minutes
